# R5

A prototype exploring the future of Radio4000. Local-first music player with client-side PostgreSQL that syncs with R4.

## Documentation

See the [`docs`](./docs) folder.

## Developing

```bash
bun install
bun run dev
```

## Deploying

The `main` branch deploys to https://pg.radio4000.com for now.
