# R5

A beta exploring the future of Radio4000. Local-first (kinda) music player with R4 as the remote server.

https://beta.radio4000.com

## Documentation

See the [`docs`](./docs) folder.

For i18n monthly batch updates, see [`docs/localization-i18n.md`](./docs/localization-i18n.md) (`i18n:extract`, `i18n:apply`, `i18n:check`).

## Developing

```bash
bun install
bun dev
```

Also see [`deployment`](./docs/deployment.md).
